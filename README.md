# NK Creative Studio

A bold, interactive portfolio website for NK Creative Studio - a creative design agency specializing in brand identity, web design, and motion graphics.

![NK Studio Preview](https://via.placeholder.com/800x400/0a0a0f/FF3CAC?text=NK+Creative+Studio)

## ✨ Features

- **Custom Animated Cursor** - Interactive cursor with ring animation
- **Hero Section** - Dynamic text animations and floating blob backgrounds
- **Work Showcase** - Grid layout featuring selected projects
- **Services Section** - Three core service offerings with hover effects
- **Statistics Display** - Key metrics and achievements
- **Testimonial Section** - Client testimonial with gradient text
- **Call-to-Action** - Contact section with email link
- **Responsive Design** - Mobile-optimized layout
- **Smooth Animations** - CSS animations and transitions throughout

## 🎨 Design Highlights

- **Color Palette**: Vibrant gradients (magenta, orange, yellow, cyan, purple, blue)
- **Typography**: Syne (display) and Outfit (body) from Google Fonts
- **Dark Theme**: Deep dark backgrounds with bright accent colors
- **Interactive Elements**: Hover effects, scale transforms, and blend modes
- **Modern Aesthetics**: Glassmorphism, noise textures, and geometric shapes

## 🚀 Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Custom properties, gradients, animations, and responsive design
- **Vanilla JavaScript** - Custom cursor interaction and smooth animations
- **Google Fonts** - Syne and Outfit font families

## 📱 Sections

1. **Navigation** - Fixed header with logo and menu links
2. **Hero** - Compelling headline with CTA buttons and scroll indicator
3. **Ticker** - Animated scrolling text showcasing services
4. **Work** - Portfolio grid with project previews
5. **Services** - Detailed service offerings
6. **Stats** - Key performance metrics
7. **Testimonial** - Client feedback
8. **Contact CTA** - Email contact information
9. **Footer** - Social links and copyright

## 🛠️ How to View

Simply open `NKstudio.html` in any modern web browser. No server or build process required.

```bash
# Open in default browser
start NKstudio.html
```

Or drag and drop the file into your browser window.
